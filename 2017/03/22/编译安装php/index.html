编译安装php

发表于 2017-03-22


多赚钱,离傻逼们远一点

安装环境

  • centos6.8 64
  • php 5.6.30

安装过程

  • 下载
    http://php.net/get/php-5.6.30.tar.xz/from/a/mirror

  • 预安装
    yum -y install libxml2 libxml2-devel

  • 解压编译
    ./configure –prefix=/usr/local/php –with-apxs2=/usr/local/httpd/bin/apxs
    make
    make install

安装完成

apache不认识xxx.php后缀
打开apache配置文件,增加
AddType application/x-httpd-php .php

#php
编译安装apache
